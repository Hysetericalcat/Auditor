# Search Skill

Perform web searches based on the given query using the tools in `websearch-utils.ts`.

## Available Tools

| Tool | Name | Purpose |
|------|------|---------|
| `webSearch` | `web_search` | Searches the web and returns relevant results |
| `webExtract` | `web_extract` | Extracts full text content from given URLs |
